╔═══════════════════════════════════════════════════════════════════════════════╗
║                          TIMING DIAGRAM                                        ║
╚═══════════════════════════════════════════════════════════════════════════════╝

Frame Processing Loop (33ms target = 30 FPS)
│
├─ 0ms:  Wait for frames (timeout: 100ms)
│        └─ Orbbec SDK frame acquisition
│
├─ 5ms:  Process depth frame
│        ├─ Convert Y16 to float32
│        ├─ Apply scale factor
│        └─ Generate colormap
│
├─ 10ms: Process IR frame
│        ├─ Convert Y8/Y16 to BGR
│        └─ Normalize values
│
├─ 15ms: MediaPipe hand detection
│        ├─ RGB conversion
│        ├─ Model inference
│        └─ Landmark extraction
│
├─ 20ms: Depth lookup & smoothing
│        ├─ Get depth at (x,y)
│        ├─ Apply EMA smoothing
│        └─ Validate interaction box
│
├─ 25ms: Gesture detection
│        ├─ Air-push check
│        ├─ Swipe detection
│        ├─ Palm detection
│        └─ Back zone check
│
├─ 28ms: State handling & actions
│        ├─ Process current state
│        ├─ Execute mode actions
│        └─ Update state if needed
│
├─ 30ms: UI update
│        ├─ Update HUD overlay
│        ├─ Render debug windows
│        └─ Process Qt events
│
└─ 33ms: Loop complete, start next frame
